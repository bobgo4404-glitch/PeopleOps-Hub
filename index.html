    PeopleOps Platform Blueprint

    인사관리 시스템을 데이터 기반 PeopleOps 플랫폼으로 고도화

    기존 핵심 메뉴는 유지하되, 사원 원장 중심의 관리 도구를 조직 운영, 인력 리스크, 정부 대응 통계, 경영 의사결정까지 연결하는 실무형 HR Tech SaaS 구조로 재설계합니다.

    가정한국 기준, 중소·중견기업 PeopleOps 운영 범위IA, 화면, 통계, 대시보드, 권한 유지 메뉴사원관리, 조직도, 증명서, 통계, 설정 주의법정 제출은 사업장별 추가 확인 필요
    1. 리브랜딩 2. UI/UX 3. IA 4. 메뉴 기능 5. 통계센터 6. 대시보드 7. 추가 기능 8. 권장안

    1. 서비스 리브랜딩 방향

    사원정보 관리에서 인력 의사결정 플랫폼으로 포지션을 전환합니다.

    서비스 컨셉

    PeopleOps Hub는 인사 원장, 조직도, 문서, 발령, 교육, 통계를 연결해 조직 운영의 현재 상태와 다음 액션을 제안하는 HR 운영 콘솔입니다.

    • Single Source of Truth
    • People Analytics
    • Actionable HR

    핵심 가치

    • 데이터 누락과 문서 누락을 조기에 발견합니다.
    • 경영진이 인력 구조와 성장·이탈 흐름을 즉시 파악합니다.
    • 정부 대응 통계와 내부 보고 지표를 같은 데이터 모델에서 산출합니다.
    • 권한과 감사 로그를 통해 개인정보 접근 리스크를 낮춥니다.

    타깃 사용자

    • 경영진: 인력 규모, 비용 영향, 조직 성장성 확인
    • 인사담당자: 입퇴사, 발령, 증명서, 통계 실무 처리
    • 부서장: 팀 인원, 계약 종료, 교육, 성과 리스크 확인
    • 운영자: 권한, 기준값, 조직 마스터, 로그 관리

    차별화 포인트

    • 사원정보 변경이 조직도, 통계, 월간보고에 즉시 반영됩니다.
    • 법정·정부 대응 가능 통계를 별도 카탈로그로 관리합니다.
    • 직무·역량·교육·성과까지 확장 가능한 People Graph를 지향합니다.
    • 운영 체크리스트와 액션 버튼을 대시보드에 배치합니다.

    2. 최신 SaaS 스타일 UI/UX 방향

    2025년형 Enterprise SaaS 기준으로 밀도, 탐색성, 데이터 해석성을 우선합니다.

    UI UX 방향 정의
    구분 방향 실행 기준
    디자인 컨셉 조용하고 밀도 있는 운영형 SaaS. 과도한 장식보다 명확한 정보 계층, 상태 배지, 액션 중심 UI를 우선합니다. 화이트·뉴트럴 배경, 8px 이하 라운드, 명확한 구분선, 색상은 상태와 우선순위 표현에 제한적으로 사용
    레이아웃 구조 좌측 사이드바, 상단 컨텍스트 바, 중앙 작업영역, 테이블·차트·상세 패널 조합을 기본으로 둡니다. 운영 화면은 검색·필터·테이블·상세/모달 구조, 분석 화면은 KPI·차트·드릴다운 테이블 구조
    사이드바 전략 기능 메뉴보다 업무 흐름 중심으로 묶습니다. 대시보드, 인사정보, 조직관리, 문서, 라이프사이클, Analytics, 설정 순으로 확장합니다. 핵심 1차 메뉴 8~10개 유지, 세부 기능은 탭·필터·상세 패널로 흡수
    대시보드 구성 역할별 뷰를 분리합니다. 경영진은 KPI와 추세, 인사담당자는 처리 대기, 부서장은 팀 리스크와 액션을 봅니다. 동일 데이터셋을 권한·역할에 따라 카드, 알림, 차트 우선순위만 다르게 노출
    카드형 UI KPI 카드는 짧은 값과 해석 문장까지 제공합니다. 카드 클릭 시 해당 필터가 적용된 상세 목록으로 이동합니다. 값, 전월 대비, 기준일, 데이터 품질 상태를 함께 표시
    데이터 시각화 인력 구성은 도넛·100% 누적 막대, 흐름은 라인·막대 혼합, 조직 비교는 정렬 가능한 막대와 피벗 테이블을 사용합니다. 다운로드, 필터 저장, 기준일 고정, 산식 설명 툴팁 제공
    모바일 대응 조회·승인·알림 확인 중심으로 축소합니다. 대량 입력, 엑셀 업로드, 설정은 데스크톱 우선으로 둡니다. 모바일에서는 KPI 1열, 테이블은 카드형 요약, 상세 액션은 하단 고정 버튼

    4. 메뉴별 상세 기능 정의

    필수 메뉴와 신규 제안 메뉴를 목적, 기능, 화면, 데이터, 권한 기준으로 정의합니다.

    사원관리
    목적
    사원 원장을 PeopleOps 플랫폼의 기준 데이터로 관리합니다.
    주요 기능
    등록·수정·검색·필터, 엑셀 업로드, CSV 다운로드, 정보완성도 점검, 개인정보 동의일 관리
    화면 구성
    검색/필터 툴바, 사원 테이블, 등록 모달, 상세 프로파일 탭, 데이터 품질 알림
    주요 데이터
    사번, 성명, 성별, 생년월일, 국적, 부서, 팀, 직책, 직급, 고용형태, 입사일, 계약종료일, 퇴직일, 교육·자격 요약
    권한 체계
    소유자·관리자 전체 수정, 인사담당 수정, 부서장 소속 조회, 조회전용 다운로드 제한 옵션
    조직도
    목적
    조직 계층과 현원을 연결해 조직별 인력 구조를 실시간으로 확인합니다.
    주요 기능
    본부·부서·팀·파트 트리, 조직 추가·이동·삭제, 조직별 인원 집계, 정원 대비 현원, 조직 변경 이력
    화면 구성
    좌측 조직 트리, 우측 조직 속성 편집, 하단 조직원 목록, 조직도 검색
    주요 데이터
    조직 ID, 조직명, 상위 조직, 조직 유형, 조직 메모, 소속 인원, 조직장, 코스트센터
    권한 체계
    조직 마스터 수정은 관리자 이상, 부서장은 소속 조직 조회, 변경 이력은 소유자·관리자 열람
    증명서 발급
    목적
    재직·경력·퇴직 증명서를 표준 템플릿으로 발급하고 이력을 남깁니다.
    주요 기능
    대상자 검색, 증명서 유형 선택, 용도·제출처 입력, 직인 이미지 반영, 출력, 발급 이력 저장
    화면 구성
    발급 입력 폼, 출력 미리보기, 발급 이력 테이블, 템플릿 설정 연결
    주요 데이터
    발급번호, 사원정보, 증명서 유형, 발급일, 용도, 제출처, 발급자, 출력 여부
    권한 체계
    발급은 인사담당 이상, 부서장은 요청 조회만, 모든 발급은 감사 로그 기록
    통계 / People Analytics
    목적
    인력 구성, 입퇴사 흐름, 정부 대응 통계, 조직 리스크 지표를 통합 제공합니다.
    주요 기능
    KPI, 차트, 교차표, 필터, 다운로드, 산식 설명, 법정 제출 대응 통계 카탈로그, 월간보고 매핑
    화면 구성
    요약 KPI, 구성 차트, 라이프사이클 추이, 정부 대응 탭, 조직별 드릴다운, 다운로드 패널
    주요 데이터
    고용형태, 성별, 연령, 근속, 입사일, 퇴직일, 조직, 직급, 직책, 사업장, 장애·보훈·외국인 여부, 휴직·휴가 이력
    권한 체계
    집계 통계는 관리자·인사담당·경영진, 민감 항목 원천 데이터는 소유자·관리자 제한
    운영설정
    목적
    회사 기준값, 사번 규칙, 조직 기준, 직인, 통계 기준을 중앙 관리합니다.
    주요 기능
    회사정보, 사번 생성 규칙, 기준 코드, 조직 기본값, 개인정보 보관 기준, 월간보고 매핑
    화면 구성
    회사 설정, 사번 규칙, 코드 목록, 증명서 직인, 데이터 초기화, 백업 안내
    주요 데이터
    회사명, 대표자, 사업자번호, 주소, HR 담당자, 직책·직급·고용형태·근무지 코드, 직인 이미지
    권한 체계
    소유자·관리자만 수정, 인사담당 일부 코드 조회, 초기화는 소유자 확인 절차 필요
    운영자관리
    목적
    개인정보 접근 권한을 최소 권한 원칙으로 통제합니다.
    주요 기능
    접근 요청 승인·반려, 역할 부여, 계정 중지, 운영 로그 조회, 권한 정책 설명
    화면 구성
    운영자 KPI, 접근 요청 목록, 계정 테이블, 감사 로그, 역할 정책 패널
    주요 데이터
    UID, 이메일, 표시명, 역할, 상태, 승인일, 마지막 수정일, 액션 로그
    권한 체계
    소유자 전용. 관리자·인사담당은 본인 세션 정보만 조회
    추가 제안 메뉴
    인재 프로파일
    스킬, 경력, 자격, 교육, 평가, 승진 이력을 사람 단위로 통합합니다.
    인재풀 관리
    채용 후보자와 내부 전환 후보자를 관리하고 채용 성과와 연결합니다.
    승계계획
    핵심 직책별 후보군, 준비도, 공백 리스크를 관리합니다.

    AI 분석
    추세 요약, 이상치 탐지, 이직 위험 신호, 교육 추천을 보조합니다. 최종 판단은 담당자가 수행합니다.

    7. 경쟁력 강화를 위한 추가 기능 제안

    PeopleOps 고도화는 데이터 품질, 분석, 예측 순서로 단계화합니다.

    P1

    인재 프로파일

    사원 원장에 스킬, 경력, 자격, 교육, 평가 요약을 연결합니다. 모든 고도화 기능의 기반입니다.

    P1

    데이터 품질 센터

    필수값 누락, 중복 사번, 조직 미매칭, 계약 종료일 누락을 자동 점검합니다.

    P1

    정부 대응 통계 카탈로그

    장애인, 고령자, 고용형태, 휴직·휴가 등 제출 검토용 통계를 기준일별로 저장합니다.

    P2

    이직 위험도 분석

    근속, 조직 변동, 계약, 평가, 교육 데이터를 결합해 위험 신호를 보조 지표로 표시합니다.

    P2

    인재 매트릭스

    성과와 잠재력을 9-box 형태로 보되 평가 편향과 개인정보 노출을 통제합니다.

    P2

    승계계획

    핵심 직책별 후보자, 준비도, 대체 가능 시점, 공석 리스크를 관리합니다.

    P2

    성과 연계 분석

    조직별 평가, 승진, 교육, 이직 흐름을 연결해 보상·육성 의사결정을 지원합니다.

    P3

    AI 분석

    월간 인력 리포트 요약, 이상치 탐지, 질문형 분석을 지원합니다. 법적 판단과 인사 결정은 사람이 확정합니다.

    P3

    인력 예측

    입퇴사 추세와 채용 계획을 기반으로 월별 재직 인원을 예측합니다. 충분한 이력 데이터가 선행되어야 합니다.

    8. 최종 권장안

    현실적인 개발 난이도와 운영 효과를 기준으로 우선순위를 정리합니다.

    최종 권장안
    구분 도입 권장도 기대효과 개발난이도 우선순위
    데이터 품질 센터매우 높음통계 신뢰도, 정부 대응, 운영 누락 방지낮음P1
    People Analytics 통계센터매우 높음경영 보고와 HR 실무 통계를 한 화면에서 처리보통P1
    정부 대응 통계 카탈로그높음장애인·고령자·고용형태 등 제출 검토 시간 절감보통P1
    역할별 대시보드높음경영진, HR, 부서장별 의사결정 속도 개선보통P1
    인재 프로파일높음교육, 평가, 승진, 핵심인재 관리 기반 확보보통P2
    승계계획·핵심인재중간핵심 직책 공백 리스크 관리높음P2
    AI 분석·이직 위험도중간이상 신호 탐지와 리포트 자동화높음P3

    사원 관리

    사원 기본정보를 등록, 수정, 검색, 필터링합니다.

    사원 등록 정보 목록
    관리
    0명
    조건에 맞는 사원이 없습니다. 검색어나 필터를 조정하거나 신규 사원을 등록하세요.

    조직도

    회사 조직 구조를 먼저 만들고, 사원을 해당 조직에 배치해 관리합니다.

    조직도

    본부, 부서, 팀, 파트, 법인 등 하위 조직을 계층형으로 구성합니다.

    부서

    조직 설정

    선택한 조직의 명칭, 상위 조직, 조직 유형을 관리합니다.

    배치 인원

    선택한 조직과 하위 조직에 배치된 인원을 확인합니다.

    증명서 발급

    재직, 경력, 퇴직 증명서 요청과 발급 이력을 관리합니다.

    발급 요청 등록

    관리자가 직접 접수하고 발급 상태를 이력으로 남깁니다.

    출력용 증명서

    작성값을 기준으로 바로 출력할 수 있는 증명서 템플릿입니다.

    입사·퇴사 처리

    입사, 수습, 퇴사 단계별 필수 확인사항과 처리 누락 위험을 점검합니다.

    근로계약, 4대보험, 퇴직 정산은 외부 신고·급여 시스템 확인이 필요합니다. 이 화면은 누락 방지를 위한 운영 체크리스트입니다.

    입사/온보딩 처리

    최근 입사자와 예정 입사자의 기본정보, 계약/서류, 연락처, 외부 신고 확인 상태를 점검합니다.

    입사 온보딩 처리 목록
    대상자 입사일 조직/직책 계약/기본정보 개인정보 동의 연락/비상연락 4대보험 처리상태

    수습 종료 예정

    수습 종료일이 있거나 곧 도래하는 인원의 평가/전환/계약 여부를 확인합니다.

    수습 종료 예정 목록
    대상자 수습 종료일 남은 기간 입사일 필수 확인 처리상태

    퇴사/계약 종료 처리

    퇴직자와 계약 종료 예정자의 계정 회수, 정산, 증명서, 보관/파기 일정을 점검합니다.

    퇴사 처리 목록
    대상자 기준일 구분 계정/장비 정산 증명서 보관/파기

    처리 누락/마감 임박

    현재 등록 데이터 기준으로 보완해야 할 항목입니다.

      표준 처리 흐름

      관리자가 확인해야 할 기본 절차입니다.

        근태/휴가 관리

        입사일과 운영 설정 기준에 따라 연차 부여 예상일수를 확인합니다.

        연차 계산은 근로기준법 제60조의 기본 구조를 단순 반영한 추정값입니다. 실제 부여일수는 출근율, 휴직, 단시간 근로, 회사 취업규칙에 따라 달라질 수 있습니다.

        연차 산정 기준

        운영 설정에서 입사일 기준 또는 회계연도 기준을 선택합니다.

        산정 기준 안내

        -

          사원별 연차 부여 예상

          재직/휴직 인원 기준으로 현재 시점의 부여 예상일수를 계산합니다.

          사원별 연차 부여 예상
          사원 입사일 근속 산정 기준일 기본 연차 1년 미만 월차 총 부여 예상 비고

          계약/발령 관리

          계약 갱신, 전보, 승진, 직책 변경을 등록하고 변경 전후 정보와 적용 여부를 관리합니다.

          발령 처리 현황

          검토, 승인, 적용, 보류 상태를 기준으로 현재 진행 상황을 확인합니다.

          계약 리스크 분류

          계약 종료일 기준으로 즉시 처리, 30일, 90일, 장기 검토 대상을 구분합니다.

          발령/계약 변경 등록

          대상자의 현재 정보를 기준으로 변경 후 값을 등록하고, 필요 시 사원정보에 바로 반영합니다.

          계약 만료/갱신 대상

          계약 종료일과 고용형태 기준으로 사전 검토가 필요한 인원입니다.

          계약 만료 및 갱신 대상
          대상자 고용형태 계약 종료일 남은 기간 검토 항목 상태 관리

          발령/계약 변경 이력

          변경 전후 값을 비교하고 사원정보 반영 여부를 관리합니다.

          발령 계약 변경 이력
          대상자 구분/적용일 변경 전 변경 후 문서 상태 관리

          처리 누락/검토 필요

          계약, 문서, 사원정보 반영 기준의 보완 항목입니다.

            운영 기준

            발령/계약 변경 시 기본 확인 흐름입니다.

              교육/자격 관리

              법정의무교육, 직무교육, 자격/면허 보유 현황을 관리합니다.

              교육 이수 현황

              자격/면허 보유

                  법정교육 이수 매트릭스

                    평가/보상

                    평가, 승진, 보상 변경 전 검토할 인사 운영 지표입니다.

                    승진/평가 검토 후보

                      보상 변경 관리 기준
